# Homebrew Tap for KrakLabs

This is the official Homebrew tap for [KrakLabs](https://github.com/kraklabs) tools.

## Installation

```bash
brew tap kraklabs/cie
brew install cie
```

Or in one command:

```bash
brew install kraklabs/cie
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `cie` | Code Intelligence Engine - semantic search and call graph analysis for your codebase |

## Updating

```bash
brew update
brew upgrade cie
```

## Troubleshooting

If you encounter issues:

```bash
# Reinstall
brew reinstall cie

# Check version
cie --version
```
